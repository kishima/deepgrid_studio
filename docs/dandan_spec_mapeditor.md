マップエディター（3Dエディットモードおよび2Dエディットモード）の**「設定メニュー」**と、そこからアクセスできる**「その他の処理」**で設定可能な項目を網羅して列挙します。

### 1. 設定メニュー（配置・構築パーツ）
ダンジョンの構造やギミック、キャラクター以外の要素をフロアに直接配置する項目です。

*   **基本属性ブロック**:
    *   **壁**: 通行不能なサイコロ状のブロック。
    *   **空（通路）**: 何もない空間。
    *   **水**: 入るとダメージを受ける液体（青）。
    *   **火（溶岩）**: 入ると大きなダメージを受ける液体（赤）。
    *   **毒**: 入るとダメージを受ける液体（緑）。
    *   **穴**: 下のフロアへ落下する空間（2Dエディットモードのみ設定可能）。
*   **ドア類（レベルごとに2種類設定可能）**:
    *   **ドア1（CLOSE / OPEN）**: 1種類目のドアの閉・開状態。
    *   **ドア2（CLOSE / OPEN）**: 2種類目のドアの閉・開状態（鉄格子など）。
*   **仕掛け・ギミック**:
    *   **鍵穴・スイッチ**: イベントを起動させるためのスイッチ。形態（鍵穴、戻らない、トグル、プッシュ）や外見を選択可能。
    *   **しかけ床**: 踏む、特定の重量が乗る、アイテムを置くなどの条件でイベントを発動させる床。
    *   **ワープポイント**: 乗ると別の座標へ転送される地点。隠し属性の設定も可能。
    *   **書ける壁**: 鉛筆アイテムでメッセージを書き込める壁。
    *   **ホロスコープブロック**: 特定の方向（南、西、上、下など全12種類）からのみ通り抜け可能な壁。
    *   **はしご**: 上下のフロアを移動するための設備。
*   **モンスター・アイテム**:
    *   **モンスター1〜4**: そのレベルに登録した4種類のモンスターを配置。
    *   **アイテム**: 作成済みのアイテム（最大199種類から選択）を配置。
*   **編集操作**:
    *   **消去**: 配置したパーツやイベント設定を削除して「壁」以外の状態（空、水など）に戻す。
    *   **現在地変更**: 2Dエディットモードで、編集作業を行う基準となる座標を移動させる。

### 2. その他の処理（システム・環境設定）
マップエディター内の「その他」メニューから設定できる、ゲーム全体の挙動や管理に関する項目です。

*   **BGM設定**: フロアごとに演奏されるBGMを全17曲（またはBGMなし）から選択。
*   **メモリー使用状況**: アイテム、モンスター、スイッチ等の配置可能数と現在の使用数を確認。
*   **イベントフラグ設定**: 64個のイベントフラグの状態（On/Off）を個別に変更・確認。
*   **スイッチ類のリセット**: 実行中のイベント停止や、使用済みの鍵穴・スイッチ、しかけ床を初期状態に戻す。
*   **連絡通路作成／削除**: レベル（階段）間を移動するための特殊な通路の設定。
*   **動作モード設定**:
    *   **移動モード**: エディット中の移動を「通常移動」か「自由移動（空中歩行可）」にするか選択。
    *   **座標表示**: 画面左上に座標と向きを表示するかどうかの切り替え。
    *   **隠し属性**: 隠し属性のついた仕掛けを表示するかどうかの切り替え。
    *   **ワープ、しかけ床**: 作業中にこれらを発動させるかどうかの設定。
*   **システム環境設定**: ゲーム速度、カーソル移動速度、BGM/効果音/足音の有無、パレット（色数）、アイテムの表示形式（簡易/詳細）など。
*   **秘薬の作成**: 液体化した魔法（秘薬）のシンボル設定と配置。
*   **LEVEL移動**: 編集するレベル（全14レベル）を切り替える。
*   **データ保存、終了**: データのセーブ、セーブして終了、セーブせずに終了の選択。
