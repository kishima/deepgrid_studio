## Charactor

『だんだんダンジョン』のキャラクターエディターで設定できる項目は、大きく分けて「名前・プロフィール」「成長・レベル」「能力パラメーター」「グラフィック」の4つのカテゴリーがあります。

出典に基づき、設定可能な全項目を列挙します。

### 1. 名前・プロフィール項目
キャラクターの個性や背景を設定する項目です。これらのデータは、プレイヤーがキャラクターに感情移入しやすくするために用意されています 。

*   **名前**: ファーストネーム（最大全角6文字／半角12文字）とセカンドネーム（名字）の両方を設定できますが、ゲーム中に表示されるのはファーストネームのみです 。
*   **性別**: 男性か女性かを選択します 。
*   **身長**: 0.0cm〜999.9cmの間で設定可能です 。
*   **体重**: 0.0kg〜999.9kgの間で設定可能です 。
*   **生年月日**: 西暦0000年〜9999年、00月00日の形式で設定します 。
*   **年齢**: 0歳〜9999歳の間で設定可能です 。
*   **好きなもの / 嫌いなもの**: それぞれ全角12文字（半角24文字）まで入力できます 。
*   **経歴**: 全角73文字（3行分）まで、キャラクターの生い立ちなどを自由に記述できます 。

### 2. 成長・レベル設定
キャラクターがどのように強くなっていくかを決める重要な項目です。

*   **レベル（初期レベル）**: 0〜99の間で、冒険開始時のレベルを設定できます 。
*   **成長タイプ**: 以下の5種類から選択でき、レベルアップ時の能力値の伸び方が異なります 。
    *   **平均型**: 直線的に普通に成長します。
    *   **早期開花型**: 初めは早く成長しますが、後半に伸び悩みます。
    *   **大器晩成型**: 前半は成長しにくいですが、後半に一気に伸びます。
    *   **天才型**: ゲーム全般にわたって能力パラメーターが高く伸び続けます。
    *   **才能なし**: 能力がほとんど成長しません。

### 3. 能力パラメーター
実戦や探索に直接影響する具体的な能力値です 。

*   **MHP（最大ヒットポイント）**: 生命力の最大値です 。
*   **MMP（最大マジックポイント）**: 魔法を使うために必要な魔力の最大値です 。
*   **攻撃力**: 武器を使用した際や攻撃時の効果に影響します 。
*   **防御力**: モンスターの攻撃を防ぐ能力です 。
*   **すばやさ**: 行動の機敏さです。数値が高いほど命中率や回避率に影響し、行動の成功率も高まります 。
*   **遠投力**: 物を遠くに投げる能力です 。
*   **運搬力**: 持ち運べるアイテムの総重量の限界値です 。
*   **肺活量**: 水中に潜っていられる時間や窒息への耐性に影響します 。
*   **耐熱力**: 火（溶岩）の空間にいる際のダメージ耐性です 。
*   **耐毒性**: 毒の空間にいる際のダメージ耐性です 。
*   **魔法知識**: 魔法を習得・使用するための知識レベルです。魔法の難易度より数値が低いと習得できません 。
*   **集中力**: 行動に集中する度合いです。「精神統一」を行うことで上昇し、攻撃のダメージ増加や行動の成功率向上に寄与します 。
*   **鑑定力**: アイテムの正体を見極める能力です 。
*   **盗みのうで**: モンスターからアイテムを盗む際の成功率に影響します 。
*   **歯の強さ**: 硬いアイテムを食べられるかどうかに影響します 。
*   **総合レベル**: 全能力パラメーターの平均値で、そのキャラクターのおおよその強さの目安になります 。

### 4. グラフィック
*   **顔のグラフィック**: キャラクターエディターからグラフィックエディターを起動し、冒険中に表示されるキャラクターの顔（肖像画）を自作して設定できます 。

## Item

アイテムエディターでは、ダンジョンに配置したりキャラクターが使用したりするアイテムについて、基本性能から特殊な属性、使用時の効果まで多岐にわたるパラメーターを設定できます,。

設定可能な項目は以下の通りです。

### 1. 基本設定
*   **アイテムの番号**: 1〜199番までの管理番号です。
*   **アイテムの名前**: 全角15文字（半角30文字）以内で設定します。
*   **グラフィック**: 床に落ちている時の標準サイズと、手やメンテナンス画面で表示される小さいサイズの2種類を作成・設定します。

### 2. 一般パラメーター（数値データ）
*   **するどさ**: 武器としての威力です。数値が高いほど攻撃時のダメージが大きくなります,。
*   **かたさ**: 防具としての性能、または食べた時の食べやすさを表します,。
*   **おもさ**: 数値×100グラムが重さになります。総重量が運搬力を超えると動けなくなります,。
*   **温度**: 照明アイテムとしての明るさや、ぶつかった時の熱ダメージに関係します,。
*   **栄養価**: 食べた時のHP回復量です。マイナスに設定するとダメージを受けます,。
*   **エントロピー**: アイティーの性質変化に関わる変数で、最大値に達すると別のアイテムに変形します,。

### 3. 属性パラメーター
*   **種別**: 一般アイテム、巻物、空容器、液体、照明、マップ、コンパス、潜望鏡、鉛筆、赤鉛筆、青鉛筆、装飾品、発光石、宝箱の14種類から選択します。
*   **大きさ**: 収納できるサイズや、容器の場合は容量（3または15）を設定します。
*   **対魔法力**: 魔法に対する抵抗力です。高いほど魔法で変化しにくくなります,。
*   **対鑑定力**: 鑑定の難易度です,。
*   **対衝撃力**: アイテムの壊れにくさです。武器などで叩かれた際の影響に関わります,。
*   **重要アイテム属性**: 「あり」に設定すると、食べたり投げたりできなくなります（鍵など）。
*   **投げやすさ**: 遠くに投げる能力や、投げた時のダメージに関係します,。
*   **持ちやすさ**: 数値が高いほど扱いやすく、攻撃の成功判定に影響します,。
*   **収納数**: 容器などの場合に、他のアイテムをいくつ収納できるか（0〜24個）を設定します,。
*   **衝突時効果**: 何かに衝突した際の音を26種類から設定できます。

### 4. 特殊設定（変形・装備）
*   **変形条件設定**: 「最大にすると変形」「月日が経つと変形」など12パターンの変化条件と、変化後のアイテムを指定できます。
*   **装備箇所**: 頭、首、体、腕、腰、足など、どこに装備できるかを複数選択して設定可能です。

### 5. アイテムの効果（増減値）
アイテムを装備した時、または食べた時にキャラクターの能力値を変化させる設定です。各パラメーターについて「-127〜127」の範囲で増分を指定できます。

*   **対象能力**: MHP、MMP、攻撃力、防御力、すばやさ、遠投力、運搬力、肺活量、耐熱力、耐毒性、魔法知識、集中力、鑑定力、盗みのうで、歯の強さ、栄養価（HP変更）。
*   **持続時間**: この効果が持続するサイクル数を設定します。0に設定すると効果は永続します。

## Enemy


モンスターエディターでは、ダンジョンに登場するモンスター（最大56種類）の名前や能力値、アルゴリズム、グラフィックなどを詳細に設定できます。

設定項目は以下の通りです。

### 1. 基本設定
*   **モンスターの番号**: 1〜56番までの管理番号です。
*   **モンスターの名前**: 全角11文字（半角22文字）以内で設定します。
*   **プロフィール**: 性格や特徴を全角21文字×2行（最大41文字）まで入力できます。

### 2. 能力値（パラメーター）
実戦での強さや耐性を設定する項目です。
*   **最大HP**: モンスターの生命力です。初期値およびダメージの限界を設定します。
*   **攻撃力**: 武器を振るったり、物にぶつかったりした際の威力です。
*   **防御力**: プレイヤーの攻撃に対する抵抗力で、0〜32767の間で設定可能です。
*   **すばやさ**: 攻撃の成功率やプレイヤーの攻撃に対する回避率に影響します（0〜32767）。
*   **攻撃頻度**: 数値が小さいほど頻繁に攻撃し、255に設定すると攻撃しなくなります。
*   **対魔法力**: 魔法に対する抵抗力で、数値が高いほど魔法の影響を受けにくくなります。
*   **体温**: 衝突時のダメージや温度変化の魔法を受けた際の影響に関係します。
*   **耐性（耐空気力・耐水力・耐熱力・耐毒力）**: 空間属性（水、火、毒など）に対する抵抗力です。128以上に設定すると、その属性の空間を好むようになります。
*   **用心深さ**: プレイヤーがモンスターからアイテムを「盗む」際の成功率に影響します。
*   **再生速度**: 死んでから復活するまでの速さです。0の場合は復活しません。

### 3. 移動・行動アルゴリズム
モンスターの動きや思考ルーチンを設定します。
*   **移動タイプ**: 「地上（通路を移動）」「空中（空間を自由に移動）」「なし（移動しない）」から選択します。
*   **はしご**: はしごを使って段差を越えて移動できるかどうかを設定します。
*   **通路**: 幅1ブロックの狭い通路を通れるかどうかを設定します。
*   **視界**: キャラクターを認識できる距離を0〜40の間で設定します。0だと認識できません。
*   **行動単位**: 移動の速さです。1が最速で、255に設定すると移動しなくなります。
*   **逃避HP**: HPがこの値を下回ると、攻撃をやめて逃げ出します。
*   **攻撃方法**: モンスターが攻撃する際のアクションを設定します。

### 4. アイテム設定
*   **所持品（1〜4）**: モンスターを倒した時に落とすアイテムや、盗めるアイテムを設定します。
*   **攻撃用（1〜4）**: モンスターが遠距離攻撃などで投げてくるアイテムを設定します。

### 5. グラフィックとアニメーション
*   **グラフィック**: 通常の状態（パターン1、2）と、攻撃用の状態（パターン3）の合計3パターンのグラフィックを自作して設定します。
*   **大きさ**: 通常サイズ（112×84ドット）の「小」と、その倍の大きさ（224×84ドット）の「大」の2種類から選択します。
*   **パターン変更**: アニメーションの切り替え速度です。数値が小さいほど速く動きます。

## Magic

魔法エディターでは、キャラクターが使用する魔法（最大80種類）について、名前や効果、使用条件、演出（グラフィック・音）などを詳細に設定できます。

設定可能な項目は以下の通りです。

### 1. 基本設定
*   **魔法の名前**: 全角6文字（半角12文字）以内で設定します。
*   **魔法のシンボル**: アクションアイコンウインドーなどで表示される魔法のマークです。グラフィックエディターで作成します。
*   **魔法の説明**: 魔法のスクロール（巻物）を「見る」コマンドで確認した際に表示される解説文です。全角20文字×2行（最大40文字）まで入力可能です。

### 2. 効果・条件パラメーター
*   **使用MP**: 魔法を唱える際に消費するMP（0〜3000）を設定します。
*   **難易度**: その魔法を習得・使用するために必要なキャラクターの「魔法知識」のレベル（0〜3000）です。知識がこの値に満たないと魔法は唱えられません。
*   **魔法種別**: 魔法の具体的な効果を27種類から選択します。
    *   **能力値変化**: MHP、MMP、攻撃力、防御力、すばやさ、遠投力、運搬力、肺活量、耐熱力、耐毒性、魔法知識、集中力、鑑定力、盗みのうで、歯の強さ、栄養価（HP）の増減。
    *   **状態変化**: MPの変更、キャラクターの復活（33%、50%、100%の3種）、照明魔法（弱、中、強の3種）など。
*   **変更値**: 魔法種別で選んだ効果の増減量（-32767〜32767）を指定します。
*   **持続時間**: 魔法の効果が続く時間をサイクル単位（0〜255）で設定します。数値を0にすると、能力値変化などの効果が永続します。

### 3. 特殊設定・演出
*   **液体化**: 魔法を液体化して空容器（ビン）に詰められるかどうか（可／不可）を設定します。
*   **光弾数**: 魔法を唱えた際にターゲットに向かって飛んでいくグラフィックの数（0〜2）を設定します。
*   **効果音**: 魔法発動時の音を26種類、または「なし」から選択します。
*   **魔法のグラフィック**: 液体化した際や光弾として飛ぶ際、または着弾時の爆発パターンとして使用するグラフィックを設定します。これにはアイテム用のグラフィックを流用します。
