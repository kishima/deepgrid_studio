『だんだんダンジョン』のイベント設定では、主に**「イベントの種類」**、実行するための**「条件（フラグや遅延）」**、そして各イベント固有の**「詳細パラメーター」**を指定できます。

詳細は以下の通りです。

### 1. イベントの種類（全14種類）
マップエディターの「鍵穴・スイッチ」や「しかけ床」などに、以下のいずれかの効果を割り当てることができます。

*   **ワープ**: キャラクターを別の座標へ瞬間移動させます。
*   **水位の変更**: 特定の場所の液体（水、火、毒）を増やしたり抜いたりします。
*   **BGM変更**: ゲーム中に流れるBGMを別の曲に切り替えます。
*   **パーティー復活**: 死んだ仲間を生き返らせ、HPを最大まで回復させます。
*   **モンスターの発生**: 指定した場所に新たにモンスターを出現させます。
*   **ドアの開閉**: 離れた場所にあるドアを開けたり閉めたりします。
*   **アイテムの発生**: 指定した場所にアイテムを出現させます。
*   **ブロックの発生**: 指定した場所に壁や空間を出現させます。
*   **デモの起動**: あらかじめ作成しておいたデモ画面（オープニング、中間デモ、エンディング）を再生します。
*   **移動モード設定**: 移動方法を「通常移動」「自由移動（空中歩行可能）」「移動不可」の間で切り替えます。
*   **スイッチの操作**: 離れた場所にあるスイッチの状態（ON/OFF）を操作します。
*   **イベントフラグ変更**: 64個あるイベントフラグの状態を変更します。
*   **イベント連結終了 / ループ設定**: イベントの動作を終了させるか、あるいは永久にループさせるかを設定します。

### 2. イベント条件の設定（共通項目）
各イベントを実行する際、以下の条件を細かく指定できます。

*   **イベント実行遅延**: イベントが発動するまでの待ち時間を、**0〜63サイクル**の間で設定できます。
*   **参照フラグ**: **0〜63番までの合計64個のフラグ**のうち、どのフラグが「On」または「Off」の時に発動するかを指定します。
*   **イベント条件結合**: 複数のフラグを参照する場合、すべての条件を満たした時に実行する「AND」か、いずれか一つを満たした時に実行する「OR」かを選択できます。

### 3. 各イベント固有の設定項目
種類に応じて、さらに詳細な数値を設定します。

*   **ワープ**: 移動先の「座標」と、ワープした直後の「向き（回転量）」を設定します。
*   **水位変更**: 発生させる液体の種類（水、火、毒、または液体を抜く）と、対象となるフロアを指定します。
*   **モンスター/アイテム発生**: 出現させる「種類」と、出現する「座標」を指定します。
*   **しかけ床独自の条件**: しかけ床の場合は、発動条件として「プレーヤーが踏む」「特定の重量以上の荷物を持って乗る」「特定のアイテムを置く」といった設定が可能です。
