# sample/portraits

サンプルプロジェクトのパーティ用ポートレート(256×256、グレースケール)。
すべて**パブリックドメイン/CC0**の銅版画・木口木版画を Wikimedia Commons から
取得し、バストアップに切り出したもの。出典・ライセンスの全記録は
[CREDITS.md](../../../../CREDITS.md)、再生成は `scripts/fetch_portraits.py`。

- `knight.png` — デューラー「騎士と死と悪魔」(1513)より騎士
- `mage.png` — レンブラント「書斎の学者(ファウスト)」(c.1652)より学者
- `priest.png` — ドレ「バルク」(聖書挿絵)より預言者
- `rogue.png` — ドレ「ユダの接吻」(聖書挿絵)より兵士の顔
- `barbarian.png` — ドレ「獅子を裂くサムソン」(聖書挿絵)よりサムソン
